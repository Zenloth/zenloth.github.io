Apps

Why

I started this as a basic & relevant project for me to begin learning JavaScript.

Background

In entertainment lighting, individual parameters of lighting fixtures are controlled on DMX channels. One DMX512 frame can contain the data for 512 channels, so when lighting rigs got more complicated, this was split into "universes". One implementation of DMX over a standard IP network, ARTNet 3, configures its universes with a hex value in the XX:X:X format (Network : Subnet : Universe), with a theoretical limit of 32,768 universes.

Objective

In essence, I wanted to create an "on-change" calculator, that converted from most standard input formats of DMX Universes to all other types (base 1, base 0, DMX channel count, ARTNet hex etc.).
